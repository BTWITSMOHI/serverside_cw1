# Alumni Management API

## Setup Instructions

1. Install dependencies:
npm install

2. Create database:
CREATE DATABASE serverside_cw1;

3. Run schema:
psql -U postgres -d serverside_cw1 -f schema.sql

4. Configure environment:
Rename .env.example to .env and fill in values

5. Start server:
npm run dev
